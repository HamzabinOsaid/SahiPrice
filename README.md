SahiPrice

SahiPrice is a simple web app that helps users compare nearby service providers and find the best prices quickly.

Features
🔍 Search for nearby providers
📍 Location-based results
💰 Compare prices easily
📱 Mobile-friendly responsive design
⚡ Fast and lightweight
🎨 Modern app-style UI
Tech Stack
HTML
CSS
JavaScript
Google Maps / Geolocation APIs
Vercel for deployment
